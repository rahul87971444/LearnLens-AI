# 🚀 LearnLens AI – Smart Study Assistant

LearnLens AI is an intelligent web application built using Django and Python that helps students analyze study material efficiently using Natural Language Processing (NLP).

---

## 🔥 Features

- 📄 Analyze pasted notes or uploaded PDFs
- 🧠 Extract important points automatically
- 📝 Generate concise summaries
- 🔑 Identify key concepts
- 📊 Generate concept diagrams
- ⬇️ Download summary as PDF
- 📱 Accessible on mobile devices

---

## 🧠 How It Works

1. User inputs notes or uploads a PDF
2. Text is processed using NLP techniques:
   - Tokenization
   - Stopword removal
   - Word frequency analysis
   - Sentence scoring
3. System generates:
   - Important sentences
   - Summary
   - Concepts
4. Users can download the summary as a PDF

---

## 🛠️ Tech Stack

- Python
- Django
- NLTK (Natural Language Processing)
- PyPDF2 (PDF text extraction)
- ReportLab (PDF generation)
- Graphviz (concept diagrams)
